# ExamSync

ExamSync is a React + TypeScript web application for school workflows, paper generation, and dashboard management.

## Local Development

1. Install dependencies:

```sh
npm install
```

2. Start development server:

```sh
npm run dev
```

3. Build for production:

```sh
npm run build
```

## Tech Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
